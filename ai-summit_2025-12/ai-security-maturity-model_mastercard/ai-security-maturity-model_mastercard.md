# Zero Trust Maturity Model Overview

- Zero Trust is a security framework that requires all users, whether inside or outside the organization's network, to be authenticated, authorized, and continuously validated before being granted access to applications and data.

- Divides the maturity model into 5 levels:
  - Level 1: Identity
  - Level 2: Devices
  - Level 3: Network
  - Level 4: Application & Networks
  - Level 5: Data

- Provides a prescriptive method to measure maturity.

## Strategy of AI Security

- Security for AI
- Security from AI
- security with AI

## What is it not?

- AI Security Posture Management
- AI Risk Management Framework
- AK Governance Framework
